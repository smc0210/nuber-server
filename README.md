# nuber-server

## Requirements

* Node.js
* Yarn ( npm or installer)
* Typescript ( yarn )
* POSTGRESQL ( official web site )
* GraphQL ( yarn )
* React

## Get Start

```bash
# package install
yarn add

psql -U postgres

# server start
yarn dev
``
